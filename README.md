# Simple collada (.dae) parser

The .tna file contains geometry and animation info.

### tna structure
- v = vertex
- vt = UV coords
- vn = vertex normal
- f = face
- w = weight of bone
- jcount = how many bones a vertex is affected by
- j = joint
- t = animation time
- a = matrix

```bash

vt: 0.9999001 9.998E-5 9.998E-5 ...
```
